# ESFramework

An Effective & Swinging framework for iOS.

## Requirements

* Xcode 5.0 or later
* iOS deployment target >= 6.0
* ARC

## Installation

There are three ways to use ESFramework in your project:

* Use [CocoaPods](http://cocoapods.org):

```ruby
pod "ESFramework"
```
